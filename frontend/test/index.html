What this test is searching for
  • Can the RL proposal add or reshape clouds over the chosen scene?
  • Does extra cloud cover move the predicted temperature toward the target?
  • Do cloud probability, radiation, humidity, and pressure change in the expected direction?
Hardcoded success criteria
  • Temperature should decrease when cloud cover increases.
  • Shortwave radiation should decrease after the cloud overlay.
  • Cloud cover and cloud probability should increase visibly.
