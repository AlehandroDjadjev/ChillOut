ChillOut Cloud RL Planner Temperature Demo

Second model showcase

Show the RL planner turning cloud inputs into cloud edits.

This page demonstrates the planned model boundary: an image mask plus the raw fourteen-feature weather vector goes in, then a generated cloud mask, action tokens, and property-map summaries come back. The original mask is fetched as a live Sentinel-2 cloud mask for the chosen place and date (falling back to a preview render when no scene is available), or you can upload your own.
