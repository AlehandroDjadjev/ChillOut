ChillOut RL Inference Runner Temperature Model

Live model runner

Run real RL weights or use an AI fallback on a live cloud mask.

Pick a place and date, upload a trained checkpoint (.pt) and optionally its stats.json, then hit Run. The checkpoint is uploaded straight to storage, the latest Sentinel-2 cloud mask is fetched for your area, and a backend worker loads the weights and runs the policy on a VM. If no checkpoint is uploaded, the backend asks OpenAI to produce an intervention overlay from the mask and target temperature: red removes clouds, green adds clouds, and yellow modifies clouds.
